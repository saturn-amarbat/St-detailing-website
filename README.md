# ST Chicagoland Mobile Detailing

Premium mobile detailing landing page built with Next.js, Tailwind CSS, React Hook Form, Zod, and Netlify Forms.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Production Build

```bash
npm run build
```

The project is configured for static export with Netlify-compatible form detection.

## Netlify

Build command:

```bash
npm run build
```

Publish directory:

```bash
out
```
